# Test UI Template Autocomplete

This is a test file to demonstrate the UI template autocomplete extension.

## Instructions

1. Place your cursor after the colon and start typing
2. Autocomplete suggestions should appear automatically
3. Try fuzzy search by typing partial words

## Examples

Try typing these patterns:

- `{{ui:` - Shows all available UI strings
- `{{ui:da` - Shows strings containing "da" (DATA, Date deployed, etc.)
- `{{ui:deploy` - Shows Deploy-related strings
- `{{ui:form` - Shows Form-related strings
- `{{ui:save` - Shows Save-related strings

## Template Examples

Click the {{ui:Deploy|bold}} button to publish your form.

Navigate to the {{ui:FORM}} tab to edit your form.

Go to the {{ui:DATA}} tab to view submissions.

Use the {{ui:Save|upper}} button to save your changes.
