| Record | soumission / formulaire | envío / formulario | “Submission” and “record” are equivalent terms (synonyms). The translation of these terms depends largely on context. Use the best translation to avoid confusion and achieve clarity, and apply that translation consistently in the specific context/ document. Translate both as “soumission” when content focuses specifically on survey records or entries in the context of data management, for example in the UI and Data table. Ex.: open, delete, and validate submissions -> ouvrir, supprimer, et valider des soumissions. Other cases when using “soumission” might be warranted include: Possible confusion with responses to individual questions or with the survey questionnaire  Modules or other content that focus specifically on data and submission management in the UI  When it is less important to identify submissions as a specific UI concept, use “réponse” or “formulaire” for submissions/responses to a survey. These are stylistically better in French for other contexts. Use these only when there is no possibility of confusion with responses to questions (for réponse) or with the survey questionnaire (for formulaire).  For example: “the enumerator uploads the submission” -> “l’enquêteur envoie le formulaire”  For example: “To modify one person’s submission” -> “pour modifier les réponses d’un individu” |
