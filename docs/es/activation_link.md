# Solución de problemas con el link de activación de la cuenta

Después de crear tu cuenta, se enviará un link de activación a la dirección de correo electrónico que proporcionaste. Debes hacer clic en este link dentro de las 72 horas para activar tu cuenta.

Si no haces clic en el link dentro de los tres días, o si el link ha expirado:
1. Intenta conectarte con el nombre de usuario/a y la contraseña que creaste. Un nuevo link de activación debería enviarse automáticamente.
2. Si eso no funciona, el problema puede ser causado por:
    * Un error tipográfico en la dirección de correo electrónico que ingresaste.
    * El dominio de correo electrónico de tu organización bloqueando nuestros mensajes.
    * El correo electrónico de activación siendo colocado en tu carpeta de spam o correo no deseado.
3. Si aún no puedes acceder a tu cuenta, por favor contáctanos en [support@kobotoolbox.org](support@kobotoolbox.org).
