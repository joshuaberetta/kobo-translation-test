# Réinitialiser votre mot de passe

Si vous avez oublié ou perdu le mot de passe de votre compte, KoboToolbox vous permet de le réinitialiser facilement.

Pour réinitialiser votre mot de passe :

-   Pour les organisations utilisant **Le serveur KoboToolbox mondial** :
    Rendez-vous sur
    [kf.kobotoolbox.org](https://kf.kobotoolbox.org)
-   Pour les organisations utilisant **Le serveur KoboToolbox Union européenne** : Rendez-vous
    sur [eu.kobotoolbox.org](https://eu.kobotoolbox.org)

![Formulaire de connexion KoboToolbox](/images/resetting_passwords/login_form.png)

-   Cliquez sur **Vous avez oublié votre mot de passe ?** sur le formulaire de connexion.
-   Saisissez votre adresse email.
-   Cliquez sur **Mise à jour du mot de passe**.

KoboToolbox vous enverra ensuite un email de réinitialisation de mot de passe après votre demande de réinitialisation du mot de passe. Si vous n'avez pas reçu d'email avec le lien de rappel du mot de passe, veuillez essayer de résoudre le problème en procédant comme suit :

-   Vérifiez si notre email de réinitialisation de mot de passe se trouve dans votre dossier spam ou courrier indésirable, car il a pu y être placé par votre fournisseur de messagerie.
-   Vérifiez si vous avez créé votre compte avec une adresse email différente - cliquez simplement à nouveau sur le lien « Vous avez oublié votre mot de passe ? » et saisissez votre adresse email alternative.
-   Il se peut également que vous demandiez à réinitialiser votre mot de passe sur le mauvais serveur. Vérifiez l'URL dans votre barre d'adresse pour vous assurer que vous êtes sur le serveur sur lequel vous avez créé votre compte,
    que ce soit [Le serveur KoboToolbox mondial](https://kf.kobotoolbox.org) ou
    [Le serveur KoboToolbox Union européenne](https://eu.kobotoolbox.org).

Si vous n'avez toujours pas reçu l'email avec le lien de réinitialisation et que vous avez essayé toutes les options de dépannage fournies ci-dessus, veuillez envoyer un email avec le sujet « Problèmes de réinitialisation de mot de passe » à
[support@kobotoolbox.org](mailto:support@kobotoolbox.org) avec votre nom d'utilisateur, l'adresse email que vous avez utilisée pour vous inscrire et le serveur sur lequel vous avez créé votre compte.
